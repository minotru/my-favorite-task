# Notes task

Implement front-end and back-end for note-taking app

*Required* functional features:

* registration/login page
* add new note
* add label to note
* search by label/note content
* group notes to folders
* support [markdown](http://commonmark.org/help/)
* send note via email
* share note with other users

*Optional* features:

* colloborative editing  (like in google docs)
* authentication via Facebook, Google, other social networks
* export to pdf
* ability to add attachments
* ability to create tables

*Required* technologies/tools/libraries

Front-end:
* React
* Redux
* [Storybook](https://github.com/storybooks/storybook)
* [styled-jsx](https://github.com/zeit/styled-jsx) or [styled-components](https://github.com/styled-components/styled-components)

Back-end:

* [express.js](https://github.com/expressjs/express) or [koa.js](https://github.com/koajs/koa)
* [mongoose](http://mongoosejs.com/)

*Optional* technologies/tools/libraries

* [redux-saga](https://github.com/redux-saga/redux-saga) or other redux middleware library
* [redux-devtools](https://github.com/gaearon/redux-devtools)
* [Immutable.js](https://facebook.github.io/immutable-js/)
* [typescript](https://github.com/Microsoft/TypeScript) or [flow](https://flow.org/)
